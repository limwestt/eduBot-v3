# EduBot V3 

 **Statut : En cours de développement (Work in Progress)**

EduBot V3 est une plateforme éducative gamifiée de type "SaaS premium". Les utilisateurs peuvent y apprendre de nouvelles compétences via des "Packs", accomplir des missions quotidiennes, évoluer sur un arbre de compétences et interagir avec une communauté. 

Le projet intègre également une Intelligence Artificielle pour proposer des défis quotidiens et valider l'expérience (XP) des utilisateurs.

##  Stack Technique
- **Backend :** Python / Django
- **Base de données :** SQLite (Développement) / PostgreSQL (Prévu pour la production)
- **Frontend :** HTML5, Vanilla JS, Tailwind CSS
- **IA :** Intégration de l'API Groq / Gemini (en cours)
